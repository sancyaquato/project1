 Простая программа на Python, которая строит графики для насосов 

Что умеет:
 Рисует график зависимости напора от подачи 
 Где давление слишком высокое срабатывает датчик отключения. 
 Учитывает статическое давление в системе 
Как запустить код и вывести график
Нужны библиотеки numpy и matplotlib
Потом просто запускаем main.py

В коде функции передаются три главных числа:
1. Максимальный расход (Q)
2. Максимальный напор (H)
3. Давление в барах  при котором насос  выключается.
